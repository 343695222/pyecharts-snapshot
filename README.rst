Installation
================================================================================

You can install it via pip:

.. code-block:: bash

    $ pip install pyecharts-snapshot


or clone it and install it:

.. code-block:: bash

    $ git clone http://github.com/chfw/pyecharts-snapshot.git
    $ cd pyecharts-snapshot
    $ python setup.py install
